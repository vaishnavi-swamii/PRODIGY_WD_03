# PRODIGY_WD_03

## Tic Tac Toe Web Application

This project is developed as part of the Prodigy InfoTech Web Development Internship.

### Features

* Interactive Tic Tac Toe Game
* Player X and Player O Turns
* Winner Detection
* Draw Detection
* Restart Game Option
* Responsive Design
* Modern Neon User Interface

### Technologies Used

* HTML
* CSS
* JavaScript
